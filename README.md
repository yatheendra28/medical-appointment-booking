# Medical Appointment Booking System

A complete Spring Boot application for managing medical appointments with doctors.

## Features

- 📋 View all appointments
- 👨‍⚕️ Browse available doctors
- 📅 Book new appointments
- ❌ Cancel appointments
- 🗑️ Delete appointments
- 📊 Dashboard with statistics
- 🎨 Responsive UI design

## Technology Stack

- **Backend**: Spring Boot 2.7.0
- **Frontend**: Thymeleaf, HTML5, CSS3, JavaScript
- **Database**: H2 (in-memory)
- **Build Tool**: Maven
- **Java Version**: 11

## Quick Start

### Prerequisites
- Java 11 or higher
- Maven 3.6+
- Git

### Clone and Run

```bash
# Clone the repository
git clone <your-repository-url>
cd medical-appointment-booking

# Build the application
mvn clean package

# Run the application
java -jar target/medical-appointment-booking-1.0.0.jar
